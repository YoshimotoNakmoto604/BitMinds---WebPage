Mining Bitcoin

Bitcoin mining is the process of adding transaction records to Bitcoin's public ledger of past transactions or blockchain. This ledger of past transactions is called the block chain as it is a chain of blocks. The block chain serves to confirm transactions to the rest of the network as having taken place. Bitcoin nodes use the block chain to distinguish legitimate Bitcoin transactions from attempts to re-spend coins that have already been spent elsewhere.

History of Bitcoin Mining

Bitcoin mining started with CPUs (Central Processing Units) of computers, then progressed to GPUs (Graphical Processing Units), and eventually evolved into ASIC (Application-Specific Integrated Circuit) miners, which are specially designed for mining Bitcoin.

Equipment for Mining

  • CPU Mining: In the early days, Bitcoin mining could be done using the CPU of a regular computer.
  • GPU Mining: Later, miners found that using the GPUs in graphics cards could achieve faster mining speeds.
  • ASIC Mining: Currently, most Bitcoin mining is done using ASIC miners, which are specialized hardware designed specifically for mining Bitcoin.
